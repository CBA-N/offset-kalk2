  PIERWSZE KROKI PO URUCHOMIENIU
================================================================

1. TEST CIĘCIA PAPIERU:
   • Formularz → Nakład: 1000
   • Format: 90×50mm (wizytówki)
   • ✅ Zaznacz: "Papier wymaga cięcia z B1"
   • Kliknij "Oblicz wycenę"
   • Sprawdź: Koszt cięcia w wynikach

2. TEST JEDNOSTEK:
   • Słowniki → Uszlachetnienia → Edytuj "Lakier UV"
   • Typ jednostki: Metrowa
   • Jednostka: "1 m²"
   • Cena: 200 PLN
   • Zapisz

3. KONFIGURACJA CIĘCIA:
   • Słowniki → Cięcie Papieru
   • Zmień roboczogodzinę (domyślnie 60 PLN)
   • Zmień wydajność (domyślnie 500 ark/h)
   • Zapisz

================================================================
  ROZWIĄZYWANIE PROBLEMÓW
================================================================

Problem: Port 7018 zajęty
Rozwiązanie:
  1. Sprawdź: lsof -i :7018
  2. Zabij proces: kill -9 <PID>
  3. Lub zmień port w app.py

Problem: Brak modułu Flask
Rozwiązanie:
  pip install flask requests

Problem: Błąd przy uruchomieniu
Rozwiązanie:
  1. Sprawdź Python: python3 --version
  2. Sprawdź pip: pip --version
  3. Zainstaluj ponownie: pip install -r requirements.txt

Problem: Pusta lista uszlachetnień
Rozwiązanie:
  Sprawdź plik slowniki_data.json - musi istnieć!
